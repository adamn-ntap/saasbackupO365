ユーザーアカウントからすべての管理者の役割を削除する

2019年8月6日寄稿者 このページのPDFをダウンロード

ユーザーのアカウントからすべての管理者の役割が削除されると、ユーザーはSaaS Backup for Office365から自動的にログアウトされます。

ステップ

  1. 設定アイコンをクリックします画面の左上にあるユーザーIDの横。

  2. [アカウント設定]をクリックします。

  3. ROLEMANAGEMENTをクリックします。

  4. 削除するユーザー名の横にある[ユーザーの削除]をクリックします。

  5. [はい]をクリックします。
